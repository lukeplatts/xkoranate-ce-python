# RP bonuses

**Bonuses** (labeled **BONUSES** in the sidebar) is a per-document
configuration for roleplay-based nation bonuses — a bump applied to a whole
nation's participants based on their forum roleplay, rather than to any one
athlete's skill. It's separate from individual athlete skills, which you set
in an event's [Signups step](building-an-event.md#2-signups).

There's one bonus configuration per document, shared across every event you
create in it.

## Settings

| Setting | Options |
|---|---|
| Assign bonuses to | **Teams** or **Participants** |
| Effect of bonus | 0–100%, how much the bonus can swing results |
| Bonus formula | **Absolute**, **Relative**, or **World Cup 36** |

Below the form is a list where you set each nation's individual bonus value.
Which formula you pick changes how those per-nation values combine with
participant skill when a competition is scorinated — if you're not sure
which to use, **Absolute** is the default and matches the original
xkoranate-CE's standard behavior.
